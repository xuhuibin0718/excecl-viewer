基于 React 和 Vite 构建的 Excel 文件预览demo，支持在线查看和编辑 Excel 文件。

## 功能特性

可与原excel文件单元格样式 、合并单元格等保持一致

## 技术栈

- React 18
- Vite
- Handsontable
- XLSX.js

## 快速开始

### 环境要求

- Node.js 16.0 或更高版本
- pnpm 包管理器

### 安装

1. 克隆仓库
```bash
git clone https://github.com/xuhuibin0718/excecl-viewer.git
cd excecl-viewer
```

2. 安装依赖
```bash
pnpm install
```

3. 启动开发服务器
```bash
pnpm dev
```

4. 构建生产版本
```bash
pnpm build
```
